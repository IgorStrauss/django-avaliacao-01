Projeto restrição de carros por pessoa

Sistema para gerenciamento de quantidade de veículos por pessoa | Cada pessoa só poderá ter até 03 veículos cadastrados em seu ID

Para esta situação, somente existem 03 modelos disponíveis [HATCH / SEDÃ / CONVERSÍVEL] e 03 cores disponíveis [AMARELO / AZUL / CINZA]

Cada pessoa pode ter até 03 veículos, porém, não pode haver repetição de modelo e de cor

Para este projeto foram utilizados: Python, Django, Postgresql, Dockerfile, Docker-compose, Poetry e Unittest.
